ApplicationConfiguration This object contains all information for setting up the SailAway application when it is first started. Other data objects will be named in this object so the application may access them if defined.
SailProfile This object contains all the information about a particular Sail that may be observed externally. It also happens to have internal information until the SailStructure object can be properly defined. Any property that can be summed up and give an external behavior to the Sail is described in its parts in SailStructure and in its sum in this object.
SailStructure This object contains all the internal structural information describing sail elements and their relationships to each other. Any property that can be summed up and give an external behavior to the Sail is described in its parts in this object and in its sum in SailProfile.
SailMission This object contains all information needed to describe the aim of a particular SailTest. Contents of this object will be used by SailAway to determine the 'closeness' of SailResult to SailMission and evaluate the value of the test.
SailEphemeris This object contains information regarding the flight of a sail that may be imported as an ephemeris file by the Satellite Tool Kit.
SailTest This object contains all information needed to describe a particular test run of a Sail. Contents of this object will be evaluated by SailAway to determine the test score after the comparison of SailResult and SailMission.
SailResult This object contains all information needed to describe the results of a particular test run of a Sail. Contents of this object will be used by SailAway to determine the test score of the related SailTest.
