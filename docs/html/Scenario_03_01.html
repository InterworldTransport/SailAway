Scenarios
Use Case Name Retrieve Sail from Library
Iteration Finished
Scenario
  1. The User indicates a desire to work with an existing Sail.
  2. The System responds by determining remote libraries are being used.
  3. The System responds by naming a library to which the retrieve attempt will be made.
  4. The System responds by sending a delivery request for the Sail to the target Library.
  5. The System waits a little while to get a receipt confirmation and then a Sail.
  6. The receipt confirmation has no errors on it, so the System proceeds silently after updating a System status log.
