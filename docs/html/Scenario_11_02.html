Iteration Finished
Scenario
  1. The User indicates a desire to run a simulation.
  2. The System responds by determining if the currently held simulation is well formed by passing the simulation to a validator.
  3. The validation process returns with a statement saying the simulation is well formed.
  4. The simulation and its command set and related Flight Assumptions list are handed to the iteration process and run until an assumption fails or the end of the command list is reached.
