Use Case Name Retrieve Sail Property Set from Library
Iteration Finished
Scenario
  1. The User indicates a desire to work with an existing Sail Property Set.
  2. The System responds by determining remote libraries are being used.
  3. The System responds by naming a library to which the retrieve attempt will be made.
  4. The System responds by sending a delivery request for the property set to the target Library.
  5. The System waits a little while to get a receipt confirmation and then the property set.
  6. The receipt confirmation arrives with errors on it.
  7. The System responds by trying other libraries sequentionally (return to step 3) until one suceeds or all fail.
  8. The User is made aware of each failure to retrieve the property set until one succeeds or all fail.
