  1. The System indicates it has new information about a flight simulation.
  2. The System responds by determining remote libraries aren't being used.
  3. The System responds by sending the current working flight simulation to the local Library.

  5. The receipt confirmation has no errors on it, so the System proceeds silently after updating a System status log.
