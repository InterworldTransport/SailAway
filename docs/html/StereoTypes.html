Use Case Stereotype Definitions

Stereotype Name Explanation Source Date
Secure A Secure process in SailAway is one in which the identities of the source and target are passed along with any other information that comes into or out of the process. It is not necessarily the case that the identifications are valid. All that can be ensured is that they are present. adiffer 2002/02/15
Communicator A Communicator is a process that must be able to interact with the Notification Tool. Unless a standard message format object is available in all environments on which SailAway runs, a Communicator must be able to perform basic formatting of messages to match the protocol of the intended notification sent to the Notification Tool. adiffer 2002/02/15
