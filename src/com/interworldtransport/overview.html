SailAway application packages:

  1. sailmodel
  2. sailevents
  3. sailgui
  4. sailli
  5. sailnti

These packages are defined as the working elements of the SailAway application. The clados package has been imported from another related project to provide all underlying mathematics one would normally use a matrix/vector/tensor library to provide.

This packages is organized as follows.

  • sailmodel: The solar sail Physical model
  • sailevent: The SailAway Event model
  • sailgui: The SailAway graphical user interface
  • sailli: The SailAway library interface
  • sailnti: The SailAway notification tool interface
