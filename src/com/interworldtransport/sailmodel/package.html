Background

To simulate solar sail structures, flights, and their optimizations, one must construct a model of the physics to have anything to simulate. The sailmodel package supports a primitive model. Future versions of sailmodel will be expanded to better represent a sail.

The sailmodel package is based on particulars that may be found in a variety of books. Sail material from Jerome Wright's book and archived JPL documents forms much of the conceptual foundation for the solar sail components of the sailmodel. The physics behind celestial mechanics and perturbation theory can be found in a number of texts starting simply at the undergraduate and progressing to the unfathomable at the graduate and higher levels. The current sailmodel implements a simplistic sail with a moderately complex perturbation model. The sailmodel can simulate a flat sail on an elliptical solar orbit not perturbed by anything other than solar radiation at present.

This package is meant to be handled by a SailAway instance, but any user interface that uses the API correctly will suffice.

Future

Future versions of the sailmodel package will include features from the following list.

  • Internal Structure classes (Finite Element Analysis)

  • Classes containing subcomponents for power, communications, and other needs

  • Other Orbit types to handle unbounded paths

  • Other Perturbers to handle alternate perturbation techniques

  • Other Force Models to handle known sources of perturbing accelerations

  • A better set of Exceptions to catch programmatic errors.
